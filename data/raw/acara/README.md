# ACARA Raw Data Files

This folder contains raw data files downloaded from the ACARA Data Access Program.
These files are NOT committed to GitHub (see .gitignore) as they are subject to
ACARA terms of use and may not be publicly republished without written consent.

- School_Profile_2008-2025.xlsx
